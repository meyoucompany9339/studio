# Web Studio — Premium Landing

Преміальний лендинг веб-студії у стилі SaaS (Stripe / Linear / Framer).

## Стек

- React 19 · Vite 6 · Tailwind CSS 4 · Lucide React · React Router

## Запуск

```bash
npm install
npm run dev
```

- **/spivpratsia** — модель співпраці, пакет, процес, FAQ, CTA
